# talent-interview-profile-poc デモ手順

同一操作列で同じ観察結果に到達するための最小手順です。

1. API と Web を起動する（ルート `README.md` の手順）。
2. Web の「人材」で **「人材を登録」** を押し、右から開くパネルで **姓・名・読み仮名（姓・名）** を入力して「登録」する。
3. 登録した人材のリンクを開く。画面は **左右ほぼ同幅**で、**左に人材・マージ済みプロフィール**、**右に面談登録・抽出・面談履歴（反映履歴の上）**が並ぶ。右側のフォームに面談テキストを入力して「面談を追加」する（履歴一覧に行が増えることを確認できる）。
4. 「テンプレート」で **「テンプレートを登録」** を押し、右パネルから必要なら YAML を追加登録する（API 起動時に **IT エンジニア用シード** `seed-it-engineer-v1` が入っている場合は省略可）。手動例は `examples/templates/default_template.yaml`。登録済み行の **確認・編集**から既存 YAML を閲覧・更新できる。
5. 人材詳細に戻り、テンプレを選んで「抽出実行」を押す。完了まで数秒以内（スタブ既定）。
6. 「プロフィール反映」を押し、マージ済みプロフィールと履歴にスナップショットが増えることを確認する。
7. 根拠として、画面上の `interview_session_id` / `extraction_run_id` / `template_version_id` を控える。
8. 任意: `GET /exports/talents/<talent_id>.json` でエクスポート JSON を取得する。
